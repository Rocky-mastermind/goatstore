## <img src="https://media2.giphy.com/media/QssGEmpkyEOhBCb7e1/giphy.gif" width="28"> &nbsp; Features

<br/>

<div align="center">

| &nbsp; | Feature | Description |
|:---:|:---|:---|
| ⚡ | **Instant Upload** | Paste JS code → get a shareable link in under a second |
| 👁️ | **View Only** | Opens directly in browser as readable code — no download |
| ⏱️ | **Auto Expire** | Every file self-destructs after 24 hours automatically |
| 🤖 | **GoatBot Native** | Use `draft.js` to upload any command file from Messenger |
| 🌐 | **Vercel KV** | Redis-powered storage — serverless & globally distributed |
| 🔒 | **Zero Auth** | Just a POST request — no login, no API key needed |

</div>

### &nbsp; `POST /upload`

```json
// Request
{
  "content": "// your javascript code here"
}

// Response
{
  "success": true,
  "id": "Cdgk1YJ-",
  "url": "https://goatstore-phi.vercel.app/raw/Cdgk1YJ-"
}
```

### &nbsp; `GET /raw/:id`

```
GET https://goatstore-phi.vercel.app/raw/:id

→ Content-Type : text/plain; charset=utf-8
→ Valid for    : 24 hours from upload
→ On expire    : 404 Not Found
```

## <img src="https://media.giphy.com/media/WUlplcMpOCEmTGBtBW/giphy.gif" width="28"> &nbsp; GoatBot — `draft.js` Command

<br/>

> 📂 Drop `draft.js` inside `modules/commands/` — then use straight from Messenger chat.

<br/>

<div align="center">

| Command | What It Does |
|:---|:---|
| `.draft info` | Reads `info.js` → uploads → sends link |
| `.draft uid` | Reads `uid.js` → uploads → sends link |
| `.draft canvas` | Reads `canvas.js` → uploads → sends link |
| `.draft` *(reply .js file)* | Uploads the attached file directly |

</div>

<br/>

```
✅ Upload Successful!

📁 File  : info.js
🔗 Link  : https://goatstore-phi.vercel.app/raw/Cdgk1YJ-
📄 Lines : 312
📦 Size  : 8.42 KB
⏱️ Expires in 24 hours
```

<br/>

```bash
# 1. Clone
git clone https://github.com/your-username/goatstore.git
cd goatstore && npm install

# 2. Setup Vercel KV
# Dashboard → Your Project → Storage → Create KV Database

# 3. Deploy
vercel --prod
```

## <img src="https://media.giphy.com/media/kdFc8fubgS31b8DsVu/giphy.gif" width="28"> &nbsp; Project Structure

<br/>

```
goatstore/
│
├── 📄  index.js          →  Main server — routing & page rendering
│
├── 📁  api/
│   ├── 📄  upload.js     →  POST /upload — saves content to KV
│   └── 📄  raw.js        →  GET /raw/:id — serves file as plain text
│
├── 📁  lib/
│   └── 📄  store.js      →  In-memory fallback store
│
├── 📄  vercel.json        →  Vercel deployment config
└── 📄  package.json
```
